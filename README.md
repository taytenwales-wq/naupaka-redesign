# naupaka – Website-Redesign (Konzept-Demo)

Ein Design-Entwurf für eine neue Website von **naupaka** (frische Poke Bowls & Salate, Bayreuth) –
als Vorschlag zur Neugestaltung der bestehenden Seite.

## Ansehen

Einfach die Datei **`index.html`** im Browser öffnen (Doppelklick).
Für die Schriftarten und die Karte wird eine Internetverbindung benötigt.

## Was steckt drin

- Reine statische Website: **HTML + CSS + JavaScript**, keine Installation nötig
- Vollständig responsiv (Desktop, Tablet, Handy)
- Flüssige Scroll- und Hover-Animationen
- Filterbare Bowl-Übersicht, „Bau dir deine Bowl"-Sektion, Nachhaltigkeit, Catering, Standort-Karte

## Struktur

```
index.html      → die komplette Seite (Inhalt, Styling, Animationen)
images/         → Logo und Foto-Assets
```

> Hinweis: Inhalte, Preise und Texte sind beispielhaft und dienen als Gestaltungsentwurf.
